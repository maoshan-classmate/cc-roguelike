# Bug: 敌人死亡后没有动画显示

**发现时间**: 2026-03-31
**优先级**: P2
**状态**: 定位中

## 复现步骤
1. 进入游戏
2. 击杀敌人
3. 观察敌人死亡时的表现

## 预期行为
敌人死亡时应该有死亡动画

## 实际行为
敌人直接消失，没有动画

## 相关代码
- 敌人死亡逻辑
- 敌人渲染逻辑

## 根因分析

**问题**: 敌人死亡时直接设置 `alive = false`，没有过渡状态和动画。

**修复方案**:
1. 服务器：敌人死亡时进入 `dying` 状态，设置 500ms 倒计时
2. 客户端：渲染 `dying` 状态敌人时显示闪烁 + 红色滤镜 + 淡出效果
