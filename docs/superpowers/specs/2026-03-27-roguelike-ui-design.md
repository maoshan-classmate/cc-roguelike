# Roguelike 游戏 UI 设计规范

## 1. 概述

本规范为局域网多人联机 Roguelike 闯关游戏定义像素复古风格的 UI 和图片资源设计。

### 1.1 项目背景

- **游戏类型**：多人联机 Roguelike 俯视角射击游戏
- **技术栈**：React + Canvas + Node.js + Socket.io
- **目标用户**：局域网联机游戏玩家

### 1.2 设计理念

参考《吸血鬼幸存者》和《像素灯塔》等经典像素游戏，采用 **经典像素工坊 (Classic Pixel Workshop)** 风格。

### 1.3 资源来源策略

- 主要使用开源像素素材（OpenGameArt.org、itch.io CC0 授权）
- 缺失资源使用 Pencil MCP 手绘像素
- 所有资源需确保可商用或开源授权

## 4. 页面设计

### 4.1 登录/注册页面 (LoginPage)

**布局**：
- 全屏深紫黑背景 `#2D1B2E`
- 中央居中白色卡片，像素风格边框
- 顶部游戏 Logo（像素风格剑与盾图标）
- 像素字体标题

**元素**：
- 游戏 Logo 图标（32x32 像素）
- 用户名输入框（像素边框样式）
- 密码输入框
- 登录/注册按钮（像素按钮风格）
- 切换登录/注册链接

**设计风格**：
- 卡片背景：`#0f3460`（深蓝）
- 按钮背景：`#4a9eff`（主色）
- 悬停效果：轻微上移 + 阴影加深

### 4.2 大厅页面 (LobbyPage)

**布局**：
- 顶部导航栏（深色卡片风格）
- 游戏 Logo + 用户信息
- 房间列表区域
- 底部操作按钮

**元素**：
- 刷新房间按钮（带刷新图标）
- 创建房间按钮（带 + 图标）
- 房间卡片：
  - 像素边框样式
  - 房间名称（像素字体）
  - 房主名 + 玩家人数
  - 状态标签（等待中绿色/游戏中黄色）

**设计风格**：
- 房间卡片：深色背景 `#16213e`
- 悬停效果：边框高亮

### 4.3 房间页面 (RoomPage)

**布局**：
- 中央居中房间信息卡片
- 顶部房间名称
- 玩家列表（4 个槽位，垂直排列）
- 底部操作按钮

**元素**：
- 玩家槽位（像素头像框 + 颜色边框）：
  - 玩家头像（像素风格 32x32）
  - 颜色边框（对应玩家颜色）
  - 玩家名称
  - 准备状态标签
  - 房主标识（皇冠图标）
- 准备按钮
- 开始游戏按钮（房主可见）
- 离开房间按钮

**设计风格**：
- 头像框：带玩家对应颜色边框
- 本地玩家：边框加粗 + 高亮

### 4.4 游戏页面 (GamePage)

#### 4.4.1 HUD 设计

**顶部 HUD 栏**：
- 楼层显示：`楼层 X/5`
- 存活玩家：`玩家 X/Y`
- 存活敌人：`敌人 X`

**玩家状态面板**：
- 像素头像
- 血条（绿色渐变，红色低血量警告）
- 玩家名称标签

**技能栏**（屏幕底部）：
- 4 个技能槽位
- 16x16 像素图标
- 数字键提示（1-4）
- 冷却遮罩（半透明深色）

#### 4.4.2 游戏结束覆盖层

**暂停菜单**：
- 半透明黑色背景 `rgba(0,0,0,0.8)`
- 像素风格按钮
- 继续游戏 / 退出游戏

**游戏结束**：
- 胜利：金色大字 + 绿色
- 失败：红色大字 + 灰色
- 返回大厅按钮

## 7. 技术实现

### 7.1 资源存储

```
src/assets/
├── images/
│   ├── ui/           # UI 资源
│   ├── characters/   # 角色精灵图
│   ├── enemies/      # 敌人精灵图
│   ├── items/        # 道具图标
│   └── tiles/        # 地牢瓦片
└── audio/            # 音效预留目录
```

### 7.2 CSS 变量扩展

在 `src/index.css` 中扩展像素风格变量：

```css
:root {
  /* 现有变量... */

  /* 像素风格 */
  --pixel-border: 4px;
  --pixel-font: 'Press Start 2P', monospace; /* 像素字体 */
}
```

### 7.3 开源素材来源

优先从以下来源获取素材：
- OpenGameArt.org (CC0/CC-BY)
- itch.io (Pixel Art 分类，筛选 CC0)
- SpriteLib
